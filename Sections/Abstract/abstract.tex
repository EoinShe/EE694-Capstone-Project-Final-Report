\begin{abstract}
        \rule{0.9\textwidth}{0.4pt} % Decorative horizontal line
\noindent % Removes indentation from the first paragraph

This report presents the design and implementation of an FPGA-based Intrusion Detection System, which has the ability for real-time detection of Denial of Service attacks. This report outlines the core design challenges encountered during development of this project and also details the solutions implemented to address them. Testing was also conducted to evaluate the system's performance, with the resulting data proving its long-term reliability and operational stability.

\vspace{1em} 
Examples of existing intrusion detection approaches are discussed, along with
relevant technical background, to provide context for the design choices made
in this project. Particular focus is given to the advantages and limitations of
implementing an IDS on FPGA hardware compared to traditional software-based
solutions.


\vspace{1em}
This project demonstrates the potential of hardware-accelerated intrusion
detection, highlighting the benefits of low-latency processing and parallel
execution for network security applications.
\end{abstract}
