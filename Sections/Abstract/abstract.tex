
A system implementing an FPGA-based Intrusion Detection System (IDS) is
developed, capable of identifying multiple network-based attack types in
real-time. The design challenges and corresponding solutions are presented
throughout this report, including packet parsing, detection logic, and
hardware-software integration. Extensive testing is performed using both
synthetic and real-world network traffic datasets, and the results are analysed
to evaluate the reliability and effectiveness of the system in detecting
malicious activity.

Examples of existing intrusion detection approaches are discussed, along with
relevant technical background, to provide context for the design choices made
in this project. Particular focus is given to the advantages and limitations of
implementing an IDS on FPGA hardware compared to traditional software-based
solutions.

This project demonstrates the potential of hardware-accelerated intrusion
detection, highlighting the benefits of low-latency processing and parallel
execution for network security applications.
